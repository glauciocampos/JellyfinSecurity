Recommended flow: Jellyfin Quick Connect. On the TV, choose Quick Connect. Then on a 2FA-verified phone or laptop, open Jellyfin and approve the displayed code. The TV will sign in and inherit the verified status — no TOTP prompt on the TV itself.

Make sure Quick Connect is enabled in Jellyfin's Dashboard → Quick Connect.

Suspicious-login detection (v1.4)

Resolve sign-in IPs to ASN + country via MaxMind GeoLite2 .mmdb files. Sign in once with a free MaxMind account, download GeoLite2-ASN and GeoLite2-Country, place the .mmdb files inside Jellyfin's config directory, and paste the absolute paths below. Without these the plugin still works — just no first-seen alerts.

Brute-Force Protection (v2.0)

Auto-bans source IPs that exceed the failure threshold across all users. LAN bypass and trusted-proxy CIDRs are always exempt. Active bans appear under the IP Bans tab.

Impossible-Travel Detection (v2.0)

Alerts when consecutive sign-ins for the same user are physically impossible given elapsed time (e.g., London then Tokyo 30 min apart). Requires a GeoLite2-City.mmdb database — without it, suspicious-login alerts still work on ASN/country alone.

WebAuthn / passkeys (v1.4)

Most setups can leave these blank — RP ID and origin are derived from the request. Override only when behind a reverse proxy whose public hostname differs from the internal one.

Tunables (v1.4)

Anyone sharing the same WAN egress (other devices on the same router, including IoT or guest WiFi) bypasses 2FA. Off by default.
